## 5. 控制流

### 5.1 If / Else

```go
// Go
if x > 0 {
    fmt.Println("positive")
} else if x == 0 {
    fmt.Println("zero")
} else {
    fmt.Println("negative")
}
```

```rust
// Rust — if 是表达式
let msg = if x > 0 { "positive" } else { "negative" };
```

```kotlin
// Kotlin — if 是表达式
val msg = if (x > 0) "positive" else "negative"
```

### 5.2 For 循环

```go
// Go — 唯一的循环关键字
for i := 0; i < 10; i++ { }
for _, v := range items { }
for condition { }
for { } // 无限循环
```

```rust
// Rust
for i in 0..10 { }
for item in &items { }
loop { } // 无限循环
```

```python
# Python
for i in range(10):
for item in items:
```

```kotlin
// Kotlin
for (i in 0 until 10) { }
for (item in items) { }
```

### 5.3 While 循环

```rust
// Rust

```

```kotlin
// Kotlin

do { } while (condition)
```

```python
# Python

```

> 注: Go 没有 while，用 `for condition { }` 代替

### 5.4 Switch / When / Match

```go
// Go
switch x {
case 1:
    fmt.Println("one")
case 2, 3:
    fmt.Println("two or three")
default:
    fmt.Println("other")
}
```

```rust

    2 | 3 => println!("two or three"),
    _ => println!("other"),
```

```kotlin

    1 -> "one"
    2, 3 -> "two or three"
    else -> "other"
}
```

### 5.5 Break / Continue / 标签

```go

    for j := 0; j < 10; j++ {
        if j == 5 { break outer }
    }
}
```

```rust
// Rust — 带标签

    for j in 0..10 {
        if j == 5 { break 'outer; }
```

## 11. 泛型

### 11.1 泛型函数

```go
// Go 1.18+
func Map[T any, U any](items []T, f func(T) U) []U {
    result := make([]U, len(items))
    for i, item := range items {
        result[i] = f(item)
    }
    return result
}
```

```rust
// Rust
fn map_items<T, U>(items: &[T], f: impl Fn(&T) -> U) -> Vec<U> {
    items.iter().map(f).collect()
}
```

```kotlin
// Kotlin
fun <T, U> List<T>.mapItems(f: (T) -> U): List<U> = map(f)
```

### 11.2 泛型类型

```go
// Go
type Stack[T any] struct {
    items []T
}
func (s *Stack[T]) Push(item T) { s.items = append(s.items, item) }
func (s *Stack[T]) Pop() T { ... }
```

```rust
// Rust
struct Stack<T> {
    items: Vec<T>,
}
impl<T> Stack<T> {
    fn push(&mut self, item: T) { self.items.push(item); }
    fn pop(&mut self) -> Option<T> { self.items.pop() }
}
```

### 11.3 泛型约束

## 13. 错误处理

### 13.1 Go 风格 — 返回 error

```go
// Go
func readFile(path string) ([]byte, error) {
    data, err := os.ReadFile(path)
    if err != nil {
        return nil, fmt.Errorf("read failed: %w", err)
    }
    return data, nil
}

data, err := readFile("test.txt")
if err != nil {
    log.Fatal(err)
}
```

### 13.2 Rust 风格 — Result + ?

```rust
// Rust
fn read_file(path: &str) -> Result<Vec<u8>, io::Error> {
    let data = fs::read(path)?;   // ? 自动传播错误
    Ok(data)
}
```

### 13.3 异常 / Try-Catch

```kotlin
// Kotlin
try {
    val data = readFile("test.txt")
} catch (e: IOException) {
    println("Error: ${e.message}")
} finally {
    cleanup()
}
```

```swift
// Swift — throws + do-try-catch
func readFile(_ path: String) throws -> Data {
    return try Data(contentsOf: URL(fileURLWithPath: path))
}
do {
    let data = try readFile("test.txt")
} catch {
    print("Error: \(error)")
}
```

### 13.4 混合风格 — 语法糖简化 Go 错误
